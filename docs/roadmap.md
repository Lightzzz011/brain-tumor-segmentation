# Project Roadmap

This document tracks the implementation roadmap for my mini project:

**Explainable Brain Tumor Segmentation and Quantitative Analysis Using TransUNet and Attention U-Net**

## Current Core Scope
- Binary tumor segmentation
- 2D slice-based training
- Models: Attention U-Net and TransUNet
- Input modalities: FLAIR + T1ce + T2
- Explainability module
- Quantitative tumor analysis
- Streamlit demo
